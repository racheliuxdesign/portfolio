Scopes Hierarchy Creator

Builds resource hierarchies quickly and boosts users' time-to-value.

Role Lead Product Designer
Duration 3 months
Team Designer, 2 Engineers, PM
Hero screenshot placeholder

Overview

Scopes define the organizational boundaries and resource groupings in the security platform. Building these hierarchies was previously a tedious, manual process. This project created a visual hierarchy builder that dramatically simplified scope creation and management.

Problem

New customers often took days or weeks to set up their resource hierarchy correctly. The existing interface required users to create each scope individually and manually define parent-child relationships through dropdown selections — an error-prone and time-consuming process that delayed time-to-value.

Before state — manual scope creation flow

Design Process

Mapped the full onboarding journey to identify where scope creation became a bottleneck. Explored tree-view, drag-and-drop, and visual canvas approaches for hierarchy building. User testing with administrators from various org sizes helped refine the interaction model. Focused on making bulk operations and templates accessible to reduce initial setup time.

Design exploration — hierarchy builder concepts
Final design — visual hierarchy creator

Outcome & Impact

The new Scopes Hierarchy Creator reduced the average setup time from days to hours. Users could visually build, modify, and reorganize their resource hierarchy through an intuitive tree interface with drag-and-drop support. Pre-built templates further accelerated onboarding for common organizational structures.
