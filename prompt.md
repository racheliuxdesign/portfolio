# Product Designer Portfolio Website

I want you to design and generate the **first version of a static Product Designer portfolio website**. This site will be hosted on **GitHub Pages**, so it must work as a **pure static site** with **no backend, no build tools, and no frameworks** unless absolutely necessary.

The purpose of this portfolio is **not SEO or public discovery**. It will be shared **only via direct link with potential employers**. Therefore the priorities are:

- Professional visual design
- Clarity of the work
- Fast loading
- Clean structure that is easy to edit later

The site must be built using **only static assets**:

- HTML
- CSS
- Vanilla JavaScript
- Images

Do **not** use heavy frameworks or dependencies such as React, Angular, Vue, Next.js, etc. The site should run directly when opened as static files and should be compatible with GitHub Pages without any build step.

The code should be **clean, readable, and easy to extend**, because we will iterate on this design later.

## Design Principles

The design language should reflect:

- Cybersecurity products
- Enterprise software
- Data-heavy interfaces
- Professional senior designer portfolio

Important constraints:

- Avoid clichés like Matrix-style falling numbers
- Avoid flashy or gimmicky effects
- The design should be modern, sleek, and professional
- Slightly dark-leaning theme is encouraged but not mandatory
- Typography should be clean and readable
- Screenshots of products should be the primary visual focus

Good reference characteristics:

- modern SaaS product marketing pages
- clean enterprise dashboards
- minimalistic design portfolios
